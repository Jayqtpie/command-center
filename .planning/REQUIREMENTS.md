# Requirements — GuidedBarakah Command Center

## R1: Project Scaffold
- Next.js 15 App Router with TypeScript and src/ directory
- Tailwind CSS v4 configured with custom design tokens
- All dependencies installed (recharts, motion, lucide-react, clsx, tailwind-merge)
- Base layout with dark mode defaults

## R2: Design System & Tokens
- Custom color palette applied via Tailwind config
- Typography scale (editorial hierarchy, not uniform)
- Spacing and layout primitives
- cn() utility (clsx + tailwind-merge)

## R3: Executive Overview Band
- Founder greeting with time-of-day awareness
- 3-4 hero metrics with editorial emphasis (not equal cards)
- Subtle entrance animations
- Above-the-fold presence

## R4: Strategic Focus Band
- Active initiatives with status indicators
- Priority ranking (not alphabetical)
- Progress visualization per initiative
- Clear visual hierarchy — primary initiative is dominant

## R5: Growth Intelligence Band
- Audience growth chart (Recharts, brand-wrapped)
- Revenue/donation trend line
- Key growth KPIs with trend indicators
- Responsive chart containers

## R6: Content Intelligence Band
- Top performing content list
- Engagement metrics per content piece
- Content type breakdown
- Performance trend indicators

## R7: Operations Visibility Band
- System health indicators
- Service status (up/degraded/down)
- Recent alerts or incidents
- Uptime percentages

## R8: Manual Input Area (Workbench)
- Quick action buttons for common founder tasks
- Notes/memo input area
- Task creation stub
- Visual separation as "your space"

## R9: Polish & Cohesion
- Smooth scroll behavior
- Consistent animation language (Motion)
- Responsive down to tablet (1024px min)
- No card soup — editorial rhythm with varied layouts
